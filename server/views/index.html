API Project: Indian Banks Search Service

User Story:

  1. GET Request to [project_url]/api/branches?q=<> and you will receive a list of banks that has that search input string across all columns (check the fields below) as JSON response.
    Example : { "branches":[ { "ifsc":"ABHY0065001", "bank_id":60, "branch":"RTGS-HO", "address":"ABHYUDAYA BANK BLDG., B.NO.71, NEHRU NAGAR, KURLA (E), MUMBAI-400024", "city":"MUMBAI", "district":"GREATER MUMBAI", "state":"MAHARASHTRA" }, { "ifsc":"ABNA0000001", "bank_id":110, "branch":"RTGS-HO", "address": "414 EMPIRE COMPLEX, SENAPATI BAPAT MARG LOWER PAREL WEST MUMBAI 400013", "city":"MUMBAI", "district":"GREATER BOMBAY", "state":"MAHARASHTRA" }, ] }
  2. GET Request to [project_url]/api/branches/autocomplete?q=<> and you will receive a list of banks that has query's branch name as JSON response.
    Example : { "branches":[ { "ifsc":"ABHY0065001", "bank_id":60, "branch":"RTGS-HO", "address":"ABHYUDAYA BANK BLDG., B.NO.71, NEHRU NAGAR, KURLA (E), MUMBAI-400024", "city":"MUMBAI", "district":"GREATER MUMBAI", "state":"MAHARASHTRA" }, { "ifsc":"ABNA0000001", "bank_id":110, "branch":"RTGS-HO", "address": "414 EMPIRE COMPLEX, SENAPATI BAPAT MARG LOWER PAREL WEST MUMBAI 400013", "city":"MUMBAI", "district":"GREATER BOMBAY", "state":"MAHARASHTRA" }, ] }
  3. If I pass an invalid URL that doesn't follow the http(s)://www.example.com(/branches?q=<>&limit=<>&offset=<>) format, the JSON response will contain an error like {"branches":[]}
